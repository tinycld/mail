## Jumping to Mail

From anywhere in the app, press **t** then **m** to jump to Mail. (The single-key shortcut **m** also works once you're inside the package.)

## In the thread list

| Shortcut | Action |
|---|---|
| **j** | Move focus to the next conversation |
| **k** | Move focus to the previous conversation |
| **Enter** | Open the focused conversation |
| **o** | Same as Enter (Gmail-style) |
| **x** | Toggle selection of the focused conversation (for bulk actions) |
| **c** | Compose a new message |

The focus indicator only appears after you've used `j` or `k` — until then, rows render without a focus ring.

## With selections

After `x`-ing one or more conversations, the top of the list shows a bulk-action toolbar — archive, trash, mark as spam, mark read / unread, apply labels. Click those buttons (no keyboard shortcuts for the actions themselves yet).

## In the compose window

| Shortcut | Action |
|---|---|
| **Cmd/Ctrl + Enter** | Send the current message |
| **Esc** | Close the compose window (drafts auto-save) |
| **Cmd/Ctrl + B** | Bold |
| **Cmd/Ctrl + I** | Italic |
| **Cmd/Ctrl + U** | Underline |
| **Cmd/Ctrl + K** | Add a link to the selected text |

## In an open thread

The thread view inherits the global shortcuts (Cmd/Ctrl + F to search, etc.) but doesn't add Mail-specific bindings. Use the toolbar buttons for archive / move / reply.
